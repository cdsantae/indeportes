Instalación del Software

Cristian Santa

  1. Ingresar a R-Project y seleccionar el sistema operativo del equipo
  2. Para Windows ingresar al link base y descargar el Software.
  3. Abrir el instalador y seguir las instrucciones Recomendadas.
  4. Para Windows ingresar al link Rtools y descargar el instalador de acuerdo con la arquitectura del sistema operativo 32-bit o 64-bit.
  5. Abrir el instalador de Rtools y seguir las instrucciones Recomendadas.
  6. Ingresar a RStudio y dirigirse a la sección All Installers.
  7. Descargar el software de acuerdo al sistema operativo del equipo.
  8. Abrir el instalador y seguir las instrucciones Recomendadas.

Una vez instalados los dos software, utilizaremos siempre RStudio

Si tiene dudas en cómo instalar los software puedes apoyarte en tutoriales de YouTube
